# Tic-Tac-Toe AI 

A professional, unbeatable Tic-Tac-Toe game built with Python and Tkinter. Play against an AI opponent powered by the **Minimax Algorithm**!

## Features

- **Beautiful GUI**: Clean, modern graphical user interface designed with Tkinter, utilizing custom colors and smooth typography.
- **Unbeatable AI**: The AI opponent uses the Minimax Algorithm, ensuring it never loses. The best you can do is a draw!
- **Score Tracking**: Keeps track of Human wins, AI wins, and Draws across multiple rounds.
- **Visual Cues**: Dynamic turn indicators, colored symbols, and winning popups.
- **Clean Architecture**: Follows Object-Oriented Programming (OOP) principles and PEP8 guidelines, making the code highly readable and modular.

## Technologies Used

- **Python 3.12**
- **Tkinter**: Standard Python library for creating the graphical user interface.
- **Minimax Algorithm**: Core decision-making algorithm for the unbeatable AI.

## Installation

1. **Clone the repository** (or download the project folder):
   ```bash
   git clone https://github.com/yourusername/TicTacToe_AI.git
   cd TicTacToe_AI
   ```

2. **Check requirements**:
   This project uses only the Python standard library. No external dependencies are required since Tkinter comes pre-installed with Python.

## How to Run

1. Open a terminal or command prompt.
2. Navigate to the project directory:
   ```bash
   cd path/to/TicTacToe_AI
   ```
3. Run the following command:
   ```bash
   python app.py
   ```
